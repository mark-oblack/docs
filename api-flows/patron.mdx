Description of API Flow.

## 1. Browse the menu

### Request

The client sends an HTTP request to retrieve dishes belonging to a specific category (e.g., Appetizer).

```shellscript
curl --location '{baseUrl}/dishes?filter=category.eq~Appetizer' \
--header 'Accept: application/json'
```

### Response

If the request was successful, the server responds with a `200 OK` status code with the response body containing the list of dishes that belong to the specified category.
